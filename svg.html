Gráficos Vectoriales Redimensionables

Con el elemento SVG (Scalable Vector Graphics) podemos crear imágenes vectoriales redimensionables de manera dinámica o interactiva utilizando diferentes atributos. Es aceptado en la mayoría de navegadores modernos.

Atributos principales: rect, circle, line, ellipse, polygon, polyline y text.

Atributos de dibujo

Rectángulo (rect)

Propiedades: x, y, rx, ry (redondeado), width, height, fill, stroke.

Círculo (circle)

Propiedades: cx, cy (centro), r (radio).

Elipse (ellipse)

Propiedades: cx, cy, rx, ry (radios horizontal y vertical).

Polígono (polygon)

Permite crear figuras cerradas uniendo puntos (parejas x,y).

Texto vectorial (text)

Permite dibujar texto que se puede seleccionar. Se puede rotar cada letra individualmente.

* Intenta seleccionar el nombre "Javier" de arriba; verás que se comporta como texto real.

Un truco útil

Si quieres mover todo un dibujo complejo sin cambiar cada coordenada, utiliza transform="translate(x,y)" al final del elemento.

<text ... transform="translate(20,5)">Javier</text>
