La propiedad transform

Con HTML5 y CSS3 podemos utilizar la propiedad 'transform' para modificar características de un <div> como la posición (position), la rotación (rotation), la escala (scale) o el -menos utilizado- el sesgo (skew).

Este cambio de posición, tamaño o rotación se produce al instante (no hay animación en este cambio, para ello habría que utilizar 'transition' o 'animation').

Para poder comprobar estas propiedades vamos a utilizar un DIV que contiene el dibujo sencillo de un martillo (creado con SVG). A este <div> le vamos a aplicar todas las transformaciones partiendo de su apariencia original.

Es muy importante tener en cuenta que para que 'transform' funcione en todos los navegadores deberemos repetir las propiedades anteponiendo los prefijos:

  • -webkit- (Chrome y Safari)
  • -moz- (Firefox)
  • -o- (Opera)
  • -ms- (Internet Explorer)

Desplazamiento (translate)

El movimiento se basa en la suma o resta de píxeles en su posición horizontal y vertical respecto a la posición en la que se encuentra el objeto en este momento. Con números positivos se desplaza hacia la derecha y hacia abajo y con números negativos hacia arriba y hacia la izquierda.

#martillo{ transform:translate(200px,5px); }

Rotación (rotate)

La rotación se basa en los grados (de 0º a 360º). El eje siempre está en el centro del <div> y se especifica añadiendo 'deg'. Valores positivos rotan a la derecha, negativos a la izquierda.

#martillo{ transform:rotate(90deg); }

Escala (scale)

Se basa en una escala horizontal seguida de la vertical, siendo 1 la escala original. Un valor de '2' representaría el 200% (el doble de tamaño).

#martillo{ transform:scale(2,1.75); }

Sesgo (skew)

Define el sesgo horizontal y posteriormente el vertical. Se utilizan grados para la deformación.

#martillo{ transform:skew(10deg, 25deg); }

Múltiples transformaciones simultáneas

Se pueden indicar tantas transformaciones como se deseen separándolas por un espacio.

#martillo{ transform:translate(200px, 0px) rotate(33deg) scale(0.5,0.5); }
