Introducción a las animaciones

De una manera parecida a las transiciones, pero con una mayor potencialidad, en HTML5 tenemos a nuestra disposición las animaciones (animation).

Primero vamos a hacer paso a paso que el martillo gire al colocar el cursor encima.

Paso a paso

1. Creamos el objeto a animar

Primero, creamos dentro de un <div> el objeto, imagen o texto que realizará la animación.

(Sitúa el cursor encima del martillo para ver la animación simple)

2. Establecemos la animación

En el código CSS indicamos qué animación tendrá que reproducir, indicando un nombre y la duración:

#martillo { animation: animacion_martillo 2s; }

Valores opcionales:

  • animation-timing-function: Define la aceleración (linear, ease, ease-in, ease-out, ease-in-out).
  • animation-iteration-count: Número de repeticiones (infinite o número).
  • animation-direction: Dirección (normal o alternate).
  • animation-delay: Retardo antes de iniciar.
  • animation-fill-mode: Estado final (none, forwards, backwards).

Ejemplo práctico complejo

Este ejemplo es más complejo, ya que mejora el movimiento para que el martillo gire de una manera más natural y clave el clavo, utilizando el parámetro infinite.

Para que el martillo baje 57 píxeles y permanezca rotado un tiempo (del 42% al 60% de la animación), el código clave es:

@keyframes animacion_martillo {
  0% { transform: rotate(0deg); top:0px; }
  42% { transform: rotate(90deg); top:57px; }
  60% { transform: rotate(90deg); top:57px; }
  100% { transform: rotate(0deg); top:0px; }
}

Compatibilidad

Todavía es necesario añadir los prefijos (-webkit-, -moz-, -o-) repitiendo todo el bloque de código @keyframes para asegurar que todos los navegadores entiendan las instrucciones.
