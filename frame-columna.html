Textos en columnas

Con HTML5 y CSS3 se pueden crear textos en columnas de una manera muy sencilla, pudiendo configurar tanto el espacio existente entre las columnas como las líneas de separación entre columnas.

Para ello utilizamos el siguiente código CSS3:

  • column-count:2; Definimos el número de columnas.
  • column-gap:10px; Definimos el espacio mínimo entre columnas.
  • column-rule:4px dotted gray; Definimos el grosor, tipo y color de la línea separadora.

Ejemplo práctico paso a paso

1. Escribimos el texto que irá en columnas

Lo primero que tenemos que hacer es escribir lo que irá en columnas dentro de un <div> que en este caso tendrá un id="columnas":

<div id="columnas">
     <li>Ataúlfo</li>
     <li>Sigerico</li>
     ...
</div>

2. Indicamos las características de las columnas

Seguidamente el código CSS haciendo referencia al id llamado 'columnas':

#columnas{
   column-count:4;
   column-gap:20px;
   column-rule:4px dotted gray;
   list-style: none;
}

    • Compatibilidad con todos los navegadores

    Como los diferentes navegadores todavía no han podido ponerse de acuerdo en una sintaxis compartida, debemos repetir el código anteponiendo los prefijos:

    #columnas{
   column-count:4;
   -webkit-column-count:4;
   -moz-column-count:4;
   -o-column-count:4;
   -ms-column-count:4;
}

    Con este código (casi) todos los navegadores que aceptan HTML5 podrán entender lo mismo y ofrecer (casi) el mismo resultado.
